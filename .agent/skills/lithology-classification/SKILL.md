# 岩性分类指南

## 一、砂泥岩判别 (自然伽马法)

| GR (API) | 岩性分类 | 说明 |
|----------|----------|------|
| < 45 | 纯净砂岩 | 泥质含量极低 |
| 45-60 | 较纯砂岩 | 泥质含量 < 15% |
| 60-80 | 泥质砂岩 | 泥质含量 15-30% |
| 80-100 | 砂质泥岩 | 泥质含量 > 30% |
| > 100 | 泥岩/页岩 | 非储层 |

## 二、密度-中子交会判别

### 砂岩特征
- 密度骨架值: 2.65 g/cc
- 中子骨架值: -0.02 v/v
- 表现: 密度孔隙度 > 中子孔隙度 (正分离)

### 石灰岩特征
- 密度骨架值: 2.71 g/cc
- 中子骨架值: 0.00 v/v
- 表现: 密度与中子孔隙度接近

### 白云岩特征
- 密度骨架值: 2.87 g/cc
- 中子骨架值: 0.02 v/v
- 表现: 密度孔隙度 < 中子孔隙度 (负分离)

## 三、特殊岩性识别

### 有机质页岩
- 高 GR (> 120 API) + 高铀含量
- 低密度 (< 2.4 g/cc) + 高中子

### 火山岩
- 低 GR (< 50 API) + 高密度 (> 2.6 g/cc)
- 低声波时差 (< 55 μs/ft)

### 煤层
- 极低密度 (< 1.8 g/cc)
- 极高中子孔隙度 (> 50%)
- 低 GR

## 四、泥质含量计算 (Vsh)

### 线性法
```
Vsh = (GR - GRmin) / (GRmax - GRmin)
```

### Larionov 老地层公式
```
Vsh = 0.33 × (2^(2×IGR) - 1)
IGR = (GR - GRmin) / (GRmax - GRmin)
```
