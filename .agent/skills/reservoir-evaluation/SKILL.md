# 储层评价指南

## 一、孔隙度计算

### 密度孔隙度
```
PHID = (ρma - ρb) / (ρma - ρf)

其中:
- ρma: 骨架密度 (砂岩2.65, 灰岩2.71, 白云岩2.87 g/cc)
- ρb: 测井密度 (RHOB)
- ρf: 流体密度 (淡水1.0, 盐水1.1, 油0.85 g/cc)
```

### 声波孔隙度
```
PHIS = (Δt - Δtma) / (Δtf - Δtma)

其中:
- Δtma: 骨架声波时差 (砂岩55.5, 灰岩47.5, 白云岩43.5 μs/ft)
- Δtf: 流体声波时差 (189 μs/ft)
```

### 有效孔隙度
```
PHIE = PHI × (1 - Vsh)
```

## 二、渗透率估算

### 经验公式 (Timur方程)
```
K = (C × PHI^a) / (Swi^b)

典型参数:
- C = 8581
- a = 4.4
- b = 2.0
```

### 科泽尼-卡曼方程
```
K = (PHI^3 × d^2) / (180 × (1-PHI)^2 × τ^2)

其中 d 为颗粒直径, τ 为迂曲度
```

## 三、储层分类标准

### 按孔渗分类

| 储层类型 | 孔隙度 (%) | 渗透率 (mD) | 储层质量 |
|----------|-----------|-------------|----------|
| I类 | > 20 | > 100 | 优质 |
| II类 | 15-20 | 10-100 | 中等 |
| III类 | 10-15 | 1-10 | 较差 |
| 非储层 | < 10 | < 1 | 差/致密 |

### 低渗-特低渗分类

| 类型 | 渗透率 (mD) | 开发难度 |
|------|------------|----------|
| 低渗 | 1-10 | 需压裂改造 |
| 特低渗 | 0.1-1 | 需大规模压裂 |
| 超低渗 | < 0.1 | 非常规开发 |

## 四、有效厚度判别

### 基本标准
- 孔隙度 > 孔隙度下限 (通常8-10%)
- 渗透率 > 渗透率下限 (通常0.1-1 mD)
- 含油饱和度 > 饱和度下限 (通常40-50%)

### 净毛比计算
```
N/G = 有效厚度 / 总厚度
```
