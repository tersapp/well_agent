# 流体识别指南

## 一、Archie 公式

### 含水饱和度计算
```
Sw = [(a × Rw) / (PHI^m × Rt)]^(1/n)

标准参数:
- a = 1.0 (胶结系数)
- m = 2.0 (孔隙度指数)
- n = 2.0 (饱和度指数)
- Rw = 地层水电阻率
```

### 含油饱和度
```
So = 1 - Sw
```

## 二、油水层判别标准

### 常规储层

| 流体类型 | Rt/Rw | Sw | RT 特征 |
|----------|-------|-----|---------|
| 油层 | > 10 | < 40% | 高电阻 |
| 油水同层 | 5-10 | 40-60% | 中高电阻 |
| 水层 | < 5 | > 60% | 低电阻 |
| 干层 | 变化大 | - | 无流体 |

### 低阻油层识别

**成因类型:**
1. 高束缚水饱和度型 (低孔低渗)
2. 高矿化度地层水型
3. 导电矿物型 (黄铁矿等)
4. 薄互层型

**识别方法:**
- 核磁共振 (T2谱)
- 阵列感应测井 (多频)
- 地层测试 (RFT/MDT)

## 三、气层识别

### 常规特征
- 密度-中子大"叉开" (挖掘效应)
- 声波时差"跳跃"
- 高电阻率

### 定量指标

| 指标 | 公式 | 气层判断 |
|------|------|----------|
| 中子-密度差 | NPHI - PHID | > 0.1 |
| 视孔隙度比 | PHIs/PHId | > 1.2 |
| 含气饱和度 | Sg = 1 - Sw - So | > 20% |

## 四、油水界面确定

### 方法
1. 电阻率突变点
2. 毛管压力曲线
3. 压力梯度法 (RFT/MDT)

### 梯度法公式
```
油柱高度 = (Pw - Po) / (ρw - ρo) / g

油水界面 = 自由水面 + 油柱高度
```
