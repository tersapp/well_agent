# 低渗透储层评价指南

## 一、分类标准

### 按渗透率分类 (国家标准)

| 类型 | 渗透率K (mD) | 孔隙度Φ (%) | 典型区块 |
|------|-------------|------------|----------|
| 常规 | > 50 | > 15 | 渤海湾 |
| 低渗 | 10-50 | 12-15 | 鄂尔多斯延长组 |
| 特低渗 | 1-10 | 8-12 | 长庆苏里格 |
| 超低渗 | 0.1-1 | 5-8 | 致密砂岩 |
| 致密 | < 0.1 | < 5 | 页岩气储层 |

## 二、储层特征

### 孔隙结构特征
- 孔喉比大 (> 100)
- 孔喉配位数低 (< 3)
- 排驱压力高 (> 1 MPa)

### 渗流特征
- 启动压力梯度存在
- 非达西渗流
- 应力敏感性强

## 三、有效性评价

### 可动流体饱和度
```
Swm = (PHI_NMR - PHI_irr) / PHI_NMR × 100%

其中:
- PHI_NMR: 核磁总孔隙度
- PHI_irr: 束缚流体孔隙度 (T2截止值以下)
```

### 储层分类依据

| 可动流体(%) | 储层级别 | 开发建议 |
|------------|---------|---------|
| > 50 | I类 | 常规开发 |
| 30-50 | II类 | 压裂改造 |
| 15-30 | III类 | 体积压裂 |
| < 15 | 差 | 暂不开发 |

## 四、开发注意事项

1. **应力敏感性**: 生产压差不宜过大
2. **速敏性**: 控制注水速度
3. **水敏性**: 注意入井液配伍
4. **毛管力**: 油水置换效率低
